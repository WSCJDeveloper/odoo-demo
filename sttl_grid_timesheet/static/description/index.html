Grid View Page

In timesheet module go to my timesheet by default, timesheets open in Grid View, showing entries by project, task, and date. You can find grid view icon at the top-right for easy tracking and timer-based logging.

Timer for Project and Task

Click the Start button from the grid view, select your project and task, and begin tracking time. When done, click Stop, optionally add a description, and hit Save to log the entry.

Project Expansion & Task Timer

Click on the projectâ€™s (+) icon to expand and view assigned tasks. You can either start the timer for a task or enter time manually directly from the grid.

On-the-Fly Project/Task Creation & Timesheet Entry

Click on the Add a line button in the grid view to create a new project or task instantly. Then, enter your hours directly without leaving the screen.

Tracking with Filters & Detailed View

Use filters to quickly locate specific entries and click the magnifying glass icon to drill down into detailed timesheet data for precise tracking and review.
